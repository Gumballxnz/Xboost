# XBoost Enterprise SaaS

## Sobre
Plataforma de automação de engajamento no Twitter/X com foco em invisibilidade e copy persuasivo.

## Stack
- **Frontend**: HTML/CSS/JS (Vanilla + Enterprise Design System)
- **Backend**: Node.js + Express (API)
- **Database**: Supabase / Redis (Queue)
- **Deploy**: Vercel

## Como Rodar
1. `npm install`
2. Configure `.env`
3. `npm start`

## Estrutura
- `public/`: Assets estáticos (Landing Page, Favicon)
- `src/`: Lógica do servidor e API
